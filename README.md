#BudgetManager

Sistema de gestión de presupuesto personal desarrollado como proyecto académico para la asignatura Teoría de Base de Datos I.
El propósito del proyecto es aplicar los conocimientos de modelado relacional, procedimientos almacenados y lógica de negocio en base de datos mediante la implementación de una solución funcional de administración financiera personal.

#Descripción

BudgetManager permite planificar y controlar las finanzas personales a través de la gestión de presupuestos, categorías, subcategorías, obligaciones fijas y transacciones financieras.
El sistema calcula la ejecución presupuestaria y facilita el análisis de ingresos, gastos y ahorros en períodos definidos.

#Tecnologías

Oracle Database

Oracle SQL Developer

Java

#Estado

Proyecto en desarrollo como parte del curso, con enfoque en la implementación de la lógica de negocio en la base de datos y una aplicación Java en consola para su interacción.
